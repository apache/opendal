## HuggingfaceConfig

### `repo_type`: `str`

Repo type of this backend. Default is model.

Available values:
- model
- dataset

### `repo_id`: `str`

Repo id of this backend.

This is required.

### `revision`: `str`

Revision of this backend.

Default is main.

### `root`: `str`

Root of this backend. Can be "/path/to/dir".

Default is "/".

### `token`: `str`

Token of this backend.

This is optional.
