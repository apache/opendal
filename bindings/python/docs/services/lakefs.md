## LakefsConfig

### `endpoint`: `str`

Base url.

This is required.

### `username`: `str`

Username for Lakefs basic authentication.

This is required.

### `password`: `str`

Password for Lakefs basic authentication.

This is required.

### `root`: `str`

Root of this backend. Can be "/path/to/dir".

Default is "/".

### `repository`: `str`

The repository name

This is required.

### `branch`: `str`

Name of the branch or a commit ID. Default is main.

This is optional.
