## WebhdfsConfig

### `root`: `str`

Root for webhdfs.

### `endpoint`: `str`

Endpoint for webhdfs.

### `user_name`: `str`

Name of the user for webhdfs.

### `delegation`: `str`

Delegation token for webhdfs.

### `disable_list_batch`: `_bool`

Disable batch listing

### `atomic_write_dir`: `str`

atomic_write_dir of this backend
