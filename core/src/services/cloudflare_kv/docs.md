- [x] list
- [ ] ~~presign~~

## Configuration

- `root`: Set the working directory of `OpenDAL`
- `api_token`: Set the token of cloudflare api
- `account_id`: Set the account identifier of cloudflare
- `namespace_id`: Set the namespace identifier of d1
