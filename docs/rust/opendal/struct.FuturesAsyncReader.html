Struct FuturesAsyncReader

Source 
pub struct FuturesAsyncReader { /* private fields */ }
Expand description

FuturesAsyncReader is the adapter of [AsyncRead], [AsyncBufRead] and [AsyncSeek] generated by Reader::into_futures_async_read.

Users can use this adapter in cases where they need to use [AsyncRead] related trait. FuturesAsyncReader reuses the same concurrent and chunk settings from Reader.

FuturesAsyncReader also implements Unpin, Send and Sync
