Layer for replacing the default HTTP client with a custom one.

This layer allows users to provide their own HTTP client implementation by implementing the HttpFetch trait. This is useful when you need to customize HTTP behavior, add authentication, use a different HTTP client library, or apply custom middleware.

ยงExamples

use opendal_core::layers::HttpClientLayer;
use opendal_core::services;
use opendal_core::Operator;
use opendal_core::Result;
use opendal_core::raw::HttpClient;

// Create a custom HTTP client
let custom_client = HttpClient::new()?;

let op = Operator::new(services::Memory::default())?
    .layer(HttpClientLayer::new(custom_client))
    .finish();
